Project: OCR and Keyword Search Web Application

Description:

This web application is designed to perform Optical Character Recognition (OCR) on uploaded images containing text in both Hindi and English. Once the text is extracted, the application translate the Hindi to English text and vice-versa and provides a basic keyword search functionality to find specific terms within the recognized content.

Features:

Image Upload: Users can upload images containing Hindi and English text.
OCR Processing: The application utilizes OCR techniques to extract text from the uploaded images.
Keyword Search: Users can enter keywords to search for specific terms within the extracted text.
Results Display: The application displays the extracted text and highlights the matching keywords.

Usage:

Upload Image: Click the "Upload Image" button and select the desired image file.
OCR Processing: The application will process the image and extract the text.
Keyword Search: Enter keywords in the search bar and press "Enter".
View Results: The extracted text will be displayed, with matching keywords.

Technologies:

Frontend: Streamlit framework.
Backend: Python, Pytesseract, Pillow, Googletrans
Deployment Platform: Render
